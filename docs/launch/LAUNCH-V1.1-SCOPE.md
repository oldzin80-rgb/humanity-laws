# Launch v1.1 Scope

v1.1 is an architecture-completion release, not a public product release.

## Included

- Clean repo structure.
- v1.0 code preserved.
- Humanity Laws book archive plan.
- Professional-boundary router.
- Source ledger.
- UI specification.
- Expanded documentation.
- Tests for safety routing and source ledger.

## Not Included

- Public app.
- Cloud database.
- User accounts.
- Billing.
- Live API integrations.
- Live medical/legal/financial/news connectors.
- Audio transcription.
- Mobile app.
- Compliance approval.

## v1.1 Exit Criteria

- TypeScript checks pass.
- Existing integration test passes.
- Professional-boundary tests pass.
- Source-ledger tests pass.
- Archive integrity test is scaffolded and fails clearly until the confirmed book source is added.
